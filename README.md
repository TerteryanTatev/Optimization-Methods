# Optimization Methods in Python

This repository contains Python implementations of classical **1D optimization methods** for finding the minimum of unimodal functions.  
It includes methods such as 
**1.Enumeration (Grid Search), \n
2.Dichotomy (Interval Halving), \n
3.Golden Section**.  \n

Each method has a separate folder with the Python script, example plots, and a brief explanation.  
This repo is intended for learning, teaching, or experimenting with numerical optimization techniques.
