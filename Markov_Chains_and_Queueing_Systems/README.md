# Markov Chains & Queueing Systems

This folder contains Python implementations for modeling stochastic processes, including discrete-time Markov Chains and basic queueing system metrics.

##  Implemented Concepts
* **Transition Probability Matrix:** Modeling state-to-state transitions.
* **Stationary Distribution:** Finding steady-state probabilities using linear algebra (eigenvectors).
* **Queueing Performance Metrics:** Calculating system utilization ($\rho$), average number of customers, and waiting times in the system and queue.

##  Technologies Used
* **Python 3**
* **NumPy:** For matrix manipulations and eigenvalue computations.
